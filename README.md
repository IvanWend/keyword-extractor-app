\# Извлечение ключевых слов (TF-IDF)

\## Описание проекта

Данное приложение реализует базовый NLP-пайплайн для \*\*извлечения ключевых слов из текста\*\* с использованием метода \*\*TF-IDF (Term Frequency – Inverse Document Frequency)\*\*.

Приложение разработано в рамках самостоятельной практики по NLP и предназначено для демонстрации:

\- работы с корпусом документов
\- вычисления TF-IDF по корпусу
\- анализа выбранного документа
\- визуализации результатов через графический интерфейс

Графический интерфейс реализован с использованием библиотеки \*\*Streamlit\*\*.

---

\## Функциональность

Приложение позволяет:

\- задать \*\*несколько документов\*\* (корпус)
\- вычислить \*\*IDF по всему корпусу документов\*\*
\- выбрать \*\*один документ для анализа\*\*
\- извлечь \*\*ключевые слова и их веса TF-IDF\*\*
\- отобразить результаты в удобном виде

---

\## Установка

Перед запуском приложения необходимо установить зависимости из файла `requirements.txt`:
pip install -r requirements.txt

\## Запуск приложения
streamlit run app.py
